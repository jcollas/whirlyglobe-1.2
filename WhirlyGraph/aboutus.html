Instructions

  • Pinch and zoom across the 3D Globe
  • Tap and hold on a country for a web view to wikipedia corresponding to the country
  • Single tap on one or more countries, then select a data set from the top left 'bar chart' icon which will display a popover. Data sets are then used to 'loft' the selected countries up to show analytical data.

This app was developed for the iOSDevCamp 2011 Hackathon.

WhirlyGraph was developed using WhirlyGlobe, an open source 3D world toolkit developed by mousebird consulting.

WhirlyGraph won the "Best International App" award during this event.
